    Documento

    Selectores

    Los selectores son bastante flexibles, y no solo son capaces de seleccionar elementos independientes, sino también diferentes elementos, dependiendo de sus tipo, elementos cercanos, contextos, atributos y estados. Algunos ejemplos:

    • p: Selecciona cualquier elemento <p>.
    • p, a: Selecciona cualquier elemento <p> o <a>.
    • .my-class: Selecciona cualquier elemento que incluya la clase my-class (class="my-class").
    • #my-id: Selecciona cualquier elemento que tenga el id my-id (id="my-id").
    • *: Selecciona todos los elementos.
    • img[src]: Selecciona cualquier elemento <img> con atributo de src
    • nav a: Selecciona cualquier elemento <a> dondequiera dentro de un elemento <nav>.
    • article > p: Selecciona cualquier elemento <p> directamente dentro de un elemento <article>.
    • h1 + p: Selecciona cualquier elemento <p> directamente después de un elemento <h1>.
    • h2 ~ h2: Selecciona cualquier elemento <h2> dondequiera después de otro elemento <h2>.
    • a:hover: Selecciona cualquier elemento <a> mientras un cursor está suspendido sobre él.

    Todos los selectores anteriores, y muchos más no mencionados pueden ser combinados y ajustados para aplicar distintos estilos estrictamente a los elementos deseados.

    Unidades

    Existen muchos tipos de unidades y valores para las propiedades de CSS, pero el tipo de unidad más usado es el de longitud. Este es definido para presentar todo tipo de elementos en el tamaño deseado, y es un factor crucial para crear una página web presentable. Las unidades de longitud más usadas son:

    • px: Equivale a 1/96 de una pulgada
    • em: Relativo al tamaño de letra del elemento padre
    • rem: Relativo al tamaño de letra del elemento raíz
    • vw: Relativo a 1% del ancho de la ventana
    • vh: Relativo a 1% de la altura de la ventana
    • vmin: Relativo a 1% de la dimensión más pequeña de la ventana
    • vmax: Relativo a 1% de la dimensión más grande de la ventana
