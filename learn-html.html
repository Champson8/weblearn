    ¿Qué es?

    El Lenguaje de Marcado de Hipertexto se utiliza para darle estructura a una página web y sus contenidos. No es un lenguaje de programación; es un lenguaje de marcado que consiste en una serie de elementos organizados de acuerdo a la manera en la que se desea presentar el contenido.

    Para comenzar, simplemente se tiene que crear un archivo HTML, cuya extensión es .html y dentro del cual irá el código.

    Anatomía

    HTML funciona con etiquetas, las cuales pueden contener otras etiquetas que dan el formato y la jerarquía deseada al documento.

    Elemento

    Como ejemplo, la creación de un párrafo:

    El elemento se puede dividir en las siguientes 3 partes:

    • <p>: Etiqueta de apertura - Es el nombre del elemento encerrado por <> e indica el inicio del elemento.
    • Mi clase de chino comienza a las 3:00 : Contenido - Es cualquier otro elemento, o en este caso, solo texto.
    • </p>: Etiqueta de cierre - Es igual a la etiqueta de apertura, pero incluye la barra de cierre / antes del nombre de la etiqueta, e indica el final del elemento.

    Los elementos tambíen pueden poseer atributos, como por ejemplo:

    Los atributos contienen información adicional sobre el elemento, la cual podría o no cambiar su apariencia o funcionamiento, dependiendo del atributo. Generalmente tienen el mismo formato; separados del nombre del elemento por espacio, y con un = separando el nombre y el valor del atributo, con el último entre comillas.

    Documento

    Cualquier página HTML requiere distintos de estos elementos para funcionar, y comúnmente se define con la misma estructura, o una similar, dondequiera que se busque:

    Dentro del elemento <head>, se incluye aquel contenido que no es directamente visible dentro de la página web, pero que aún así la puede afectar de varias otras maneras. Puede incluir especificaciones del título, ícono, estilos CSS o códigos de Javascript.

    En cambio, dentro del elemento <body>, se incluye aquel contenido que sí se desea mostrar dentro de la página web. Puede incluir texto u otros elementos como párrafos (<p>), listas (<ul>), imágenes (<img />), etc.

    Marcado

    El marcado de texto es importante para especificar qué significan aquel contenido que sirve como texto en una página.

    El elemento de imagen tiene dos atributos cruciales para su funcionamiento:

    • src: Especifica la ruta o ubicación del archivo que se desea mostrar, como un URL externo o archivo local.
    • alt: Es el texto que se muestra en lugar de la imagen, si es que esta falló al cargar o está en el proceso.

    Vínculos

    Los vínculos crean enlaces a distintos lugares de la web, los que podrían ser archivos, páginas dentro del mismo sitio, u otros sitios web por completo. Estos requieren contenido dentro de ellos, ya sea texto o imágenes, ya que necesitan ser visibles para navegar la web al interactuar con ellos.

    El único atributo obligatorio que un enlace posee es href, el cual especifica la dirección local o URL externo a la que se desea llevar cuando se interactúa con el elemento.
