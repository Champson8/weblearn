    ¿Qué es?

    JavaScript es un lenguaje de programación que le da la mayoría de la interactividad y contenido dinámico a una página web. Esto desbloquea el 90% de todas las posibilidades que se pueden llevar a cabo en una página web, desde botones, juegos, aplicaciones, modelos, etc.

    Variables

Tambíen se puede usar la palabra clave var en lugar de let, pero no es recomendado. Similarmente, let se puede sustituir por const, para especificar que el valor de esa variable no podrá ser cambiado después de su declaración.

    Es importante notar que toda declaración como esta deberá de llevar un punto y coma (;) donde termina. Esto es para delimitar el rango de cada declaración y que diferentes delcaraciones no conflictúen entre sí.

    Para poder leer el valor de una variable, generalmente se usa el comando de console.log(), lo que registra cualquier variable, valor o expresión dentro de los paréntesis () a la consola del navegador.

    Tipos

    Cada variable que almacena información se puede clasificar en un tipo, el cual define qué tipo de datos son los almacenados y cómo se puede interactuar con ellos. Estos son algunos de los tipos más comunes:

    • String: Secuencia de texto, escrita entre comillas ("", '' o ``).
    • Number: Cualquier número, sin comillas.
    • Boolean: Valor verdadero o falso, escrito como true o false.
    • Array: Uno o más valores de cualquier tipo ordenados en una lista. (["Texto", 5, false] )
    • Object: Casi cualquier cosa, ya que JavaScript maneja todo tipo de datos como objetos. Contienen parejas de propiedades y sus valores. ({"prop1": "Texto", "prop2": true})

    Operadores

    Un operador funciona como una operación matemática, o en algunos casos, de comparación, la cual siempre devuelve el valor resultante de la operación. Los operadores básicos son:

    • +: Suma o concatenación - Suma dos números o une dos cadenas de texto.
    • -: Resta
    • *: Multiplicación
    • /: División
    • =: Asignación - Asigna un valo a una variable.
    • ===: Igualdad - Comprueba si dos valores son iguales, devolviendo un Boolean.
    • !: Negación - Cambia un valor equivalente a true a false, y viceversa.
    • !==: Diferencia - Comprueba si dos valores no son iguales, devolviendo un Boolean.
